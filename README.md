# Task Manager CLI

## Prerequisites

- Python 3.6 or higher installed on your machine.
- No external libraries are required (uses built-in datetime).

## Installation

- Clone or Download: Save the code provided in a file named app.py.
- Open Terminal: Navigate to the directory where you saved the file.

## Running the Application

- To launch the program, run the following command in your terminal:
  - python app.py
